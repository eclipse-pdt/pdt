Using JavaScript

These procedures describe how to access JavaScript Code Assist options from within your PHP files, and how to view your JavaScript objects.

 

 

To access JavaScript Code Assist options:

  1. Type the relevant HTML and JavaScript tags:

  1. JavaScript Code Assist and relevant syntax coloring will now be available.
    Type the first few letters of a JavaScript string and press Ctrl+Space to activate Code Assist.
    JavaScript Code Assist options will be preceded by either a blue triangle , indicating a reserved JavaScript word, object or variable, or a green circle , indicating a function.
    Select the relevant option from the Code Assist window by double-clicking or pressing Enter.

  2. If you selected a JavaScript class, type a period "." after the name of the class to display a Code Assist window with the classes' relevant functions and methods.

  1. Select the required option to complete your JavaScript code.

 

JavaScript objects and elements can be viewed within the HTML section of the Outline view.

 

 

To view your JavaScript objects in the Outline view:

  1. Go to the Outline view.

  2. If your JavaScript objects are contained within a PHP file, click the Menu arrow on the Outline view's toolbar and select html.
    HTML and JavaScript objects contained within the file will be displayed in a tree view.

  1. Double-Clicking the <Script> node in the outline view will select the entire <script> element in the Editor.
