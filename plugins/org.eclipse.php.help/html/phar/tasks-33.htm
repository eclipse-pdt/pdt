Creating a New PHAR File

To create a new PHAR file in the workbench:

  1. In the Package Explorer, you can optionally pre-select one or more PHP elements to export. (These will be automatically selected in the PHAR Package Specification wizard page, described in Step 4.)
  2. Either from the context menu or from the menu bar's File menu, select Export.
  3. Expand the PHP node and select PHAR file. Click Next.
  4. In the PHAR File Specification page, select the resources that you want to export in the Select the resources to export field.
  5. Select the appropriate checkbox to specify whether you want to Export generated class files and resources or Export Java source files and resources. Note: Selected resources are exported in both cases.
  6. In the Select the export destination field, either type or click Browse to select a location for the PHAR file.
  7. Select or clear the Compress the contents of the PHAR file checkbox.
  8. Select or clear the Overwrite existing files without warning checkbox. If you clear this checkbox, then you will be prompted to confirm the replacement of each file that will be overwritten.
  9. Note: The overwrite option is applied when writing the PHAR file, the PHAR description, and the manifest file.
  10. You have two options:
    • Click Finish to create the PHAR file immediately.
    • Click Next to use the PHAR Packaging Options page to set advanced options, create a PHAR description, or change the default manifest.
