Regenerating a PHAR File

You can use a PHAR file description to regenerate a previously created PHAR file.

  1. Select one or more PHAR file descriptions in your workbench.
  2. From the selection's pop-up menu, select Create PHAR. The PHAR file(s) are regenerated.
